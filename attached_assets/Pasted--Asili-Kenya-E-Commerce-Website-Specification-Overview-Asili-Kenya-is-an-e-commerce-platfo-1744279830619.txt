# Asili Kenya E-Commerce Website Specification

## Overview

**Asili Kenya** is an e-commerce platform connecting buyers to authentic Kenyan-made products across various categories. The platform focuses on simplicity, modernism, and professionalism, promoting local artisans and manufacturers.

---

## 1. Homepage Features

- **Hero Section (Top of the page):**
  - Rotating banner showcasing:
    - Best-selling products
    - Promotions
    - Discounts (if applicable)
    - Featured categories or seasonal items

- **Left Sidebar Navigation Menu:**
  - Toggleable menu
  - Contains product categories:
    - Fashion
    - Foods & Drinks
    - Crafts
    - Gifts
    - Technology
    - Health & Beauty
    - Furniture
    - Textiles
    - Plants
    - Jua Kali

- **Main Product Display Area:**
  - Grid layout displaying featured and trending products
  - Each product shows:
    - Product name
    - Thumbnail image
    - Price

- **Right-Side Cart Icon:**
  - Displays the number of items added to cart
  - Always visible on all pages

- **Footer Section:**
  - Contact information
  - Social media icons (Facebook, Instagram, Twitter, WhatsApp)
  - Brief mission tagline

---

## 2. Product Page

- **Product Gallery:**
  - Up to 5 images per product
  - Image zoom or lightbox feature for viewing

- **Product Information Section:**
  - Product name
  - Category
  - Description
  - Price

- **Order Section (Right Side):**
  - Quantity selector (only allows if in stock)
  - Total price auto-calculation
  - “Add to Cart” button
  - “Checkout” button

---

## 3. Checkout Flow

- **When Checkout Button is Clicked:**
  - Redirects to a simple **Contact Form Page**

- **Contact Form Fields:**
  - Full Name
  - Phone Number
  - Delivery Address
  - Notes/Instructions (optional)
  - Submit Order

- After submission, show a confirmation message (e.g., "Thank you for your order. We'll contact you shortly.")

---

## 4. Admin Panel (You Only)

- **No Seller Accounts**
- Only admin (you) can upload and manage products
- Product entry includes:
  - Product name
  - Up to 5 images
  - Description
  - Price
  - Category
  - Stock Quantity

---

## 5. User Accounts

- **No buyer registration/login for now**
- Orders are placed via contact form

---

## 6. Technical Features & Design Notes

- **Simple and responsive UI**
- Clean, minimal, and modern design
- Mobile-first optimisation
- Secure cart session
- Product sorting/filtering within categories (optional future feature)
- SEO-friendly structure

---

## 7. Optional Future Upgrades

- Add buyer accounts
- Add seller dashboard
- Payment gateway integration
- Order tracking system
- Admin dashboard with sales analytics

---

## 8. Tech Stack Recommendation (for Developer)

- **Frontend**: React / Vue.js (or HTML/CSS/JS for simplicity)
- **Backend**: Node.js + Express / Django / Laravel
- **Database**: MongoDB / MySQL
- **Hosting**: Vercel / Netlify (for frontend) + Firebase / DigitalOcean / AWS Free Tier
- **Image Storage**: Cloudinary or Firebase Storage
- **Form Handling**: Email via Formspree, Nodemailer, or Firebase Functions
